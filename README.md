## Watcher
an tui to check how much u suck at leauge of legends.
## How to use

```mermaid
flowchart LR;
A[open app]--> B[type name of lauge of legends accound]
B--> C[choose region]
C--> D[press `Enter`]
D--> E[see how much you suck at this game ]
```




## Installation 
#### linux:
```bash 
git clone https://github.com/Horryportier/Watcher
cd Watcher
chmod +x install.sh 
./install.sh
```
